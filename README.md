# Week 08: Project 03: Final Project - A Full Project Proposal - Student: Rodrigo Delacotera
This is my final project for course CMPA-3301, at Texas Tech University.

## Purpose of this website
- To demonstrate basic HTML knowledge, including HTML structure and CSS skills that I have learned as part of my Computers Applications course CMPA-3301. It will include the following files: index.html, about.html, information.html and style.css.
- To demonstrate general project management knowledge, as studied on our textbook "Project Management for The Unofficial Project Manager" by Kory Kogon and Suzette Blakemore. (Updated and Revised Edition, 2024). This knowledge includes development of different stages of a project, including scoping, planning, engaging the team, tracking and adapting, and finally, closing the project. In order to showcase this knowledge, I will develop several documents, as required by this project: scope.md, plan.md and retrospective.md.
- To share medical information regarding common medical problems like diabetes mellitus and hypertension, so lay people, with no prior medical knowledge may understand how to identify these issues and know when to seek timely medical attention.

## Process
I first created the public GitHub repository named Project-03. All the HTML and CSS files were uploaded, including also the PNG files. Later on, I uploaded all the project management documentation, including scope.md, plan.md and retrospective.md. The website was published to GitHub pages.
The next step was to create an experimental branch, so I could have a safe space to test several styling features, without affecting the main branch. Once all features and styles were working appropriately, I merged the experimental branch into the main branch.
A formal closing out and proofread of the entire document was done before submission.
